# DSA Repository

This repository contains various **Data Structures and Algorithms (DSA)** codes in **C++**.  
It is organized by topics and folders to make it easier to navigate and study.

---

## Folder Structure

- **Array/** : All array-related problems and solutions  
- **LinkedList/** : Linked List problems and implementations  
- **Stack/** : Stack implementation and related problems  
- **Queue/** : Queue implementation and related problems  
- **Recursion/** : Recursion-based problems  
- **Searching/** : Searching algorithms  
- **Sorting/** : Sorting algorithms  

> Add more folders as needed for new topics.

---

## Usage

1. Clone this repository to your local machine:
\`\`\`bash
git clone https://github.com/ansarulhaque1025/DSA.git
\`\`\`

2. Navigate to the desired folder:
\`\`\`bash
cd DSA/Array
\`\`\`

3. Open and run the C++ code in any IDE or compiler.
